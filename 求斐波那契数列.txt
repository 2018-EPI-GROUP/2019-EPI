1，求Fibonacci数列
Fibonacci数列有以下特点，其第一第二项除外1，从第三个数开始，该数是前两个数的和，即：
                        1 n = 1
                F（n）= 1 n = 2
                        F（n-2）+ F（N-1）n> 2
求小于n的最大Fibonacci数，代码如下：
#include <stdio.h>

int main（void）{
    int n = 0;
    int tmp = 0;
    int f1 = 1;
    int f2 = 1;
    
    printf（“请输入n：”）;
    scanf（“％d”，＆n）;
    
    / *求小于n的最大fibonacci数* /
    而（f2 <2）{
        / * f1为上一个fibonacci数，f2为本次得到的fibonacci数* /
        tmp = f2;
        f2 = f1 + f2;
        f1 = tmp;
    }
    / * f1为小于n的最大fibonacci数，/ f2为大于或等于n的最小fibonacci数* /
    printf（“小于％d的最大斐波那契数是％d。\ n”，n.f1，）;
    
    返回0;
 }

           请输入n：14
           小于14的最大数量为13。
